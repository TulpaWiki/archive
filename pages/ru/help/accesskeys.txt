====== Комбинации клавиш ======
Для более быстрой навигации и редактированию текста вы можете использовать комбинации клавиш в «[[:tulpawiki|TulpaWiki]]». К сожалению, комбинации клавиш могут отличаться в разных операционных системах и браузерах. Обычно нужно нажать «Alt» и клавишу. На «Макинтошах» --- «Ctrl» и клавишу. В «Опере», поскольку она уже использует навигацию с помощью клавиатуры, вы должны переключиться в специальный режим, нажав «Shift+Esc».

Чтобы понять, какую комбинацию клавиш вам нужно нажимать, просто попробуйте разные варианты:

  * Alt+Shift+H
  * Alt+H
  * Ctrl+H
  * Alt+H, потом нажмите Enter
  * Ctrl+H, потом нажмите Enter
  * Shift+Esc, потом нажмите H

Alt+H обычно открывает справку во многих браузерах.

**Подсказка:** В Safari 4+ вам, возможно, придётся использовать "Ctrl+Alt+key".

**Подсказка:** в Opera 8.51 вы должны нажать Shift+Esc, затем нажмите клавишу, а затем нажмите клавишу "Enter". Для всех остальных горячих клавиш воспользуйтесь "H" и "F". В [[http://www.opera.com/|Opera]] 9.0 процесс полегче: нажать "Shift+Esc", затем -- //горячая клавиша//. 

**Подсказка:** [[http://www.mozilla.com/en-US/firefox/|Firefox 2]] использует "Alt+Shift+Key" для обхода конфликтов с горячими клавишами браузера. Для восстановления старого поведения на Windows, напишите ''about:config'' в адресной строке, найдите параметр ui.key.contentAccess и установите его значение на **4**

**Подсказка:** Для «Google Chrome» используйте: Alt+Shift+Клавиша

===== Общие клавиши =====
Клавиши, всегда доступные в «TulpaWiki»:

^  Буква  ^  Действие  ^
|  ''E''  | Редактировать текущую страницу  |
|  ''V''  | Показать исходный код страницы (если кнопка\\ «Показать исходный код» (View Pagesource) отображается)|
|  ''O''  | Показать историю изменений текущей страницы  |
|  ''R''  | Показать последние изменения в вики  |
|  ''F''  | Поместить курсор в поле поиска  |
|  ''H''  | Открыть стартовую страницу   |
|  ''X''  | Показать страницу с перечнем всех страниц вики  |


===== Режим редактирования =====
Клавиши, доступные в режиме редактирования:

^  Буква  ^  Действие  ^
|  ''B''  | **Полужирный текст**  |
|  ''I''  | //Наклонный текст//  |
|  ''U''  | __Подчёркнутый текст__  |
|  ''C''  | ''Монощиринный''  |
|  ''D''  | <del>Зачёкнутый текст</del>  |
|  ''1''  | Заголовок 1-го уровня |
|  ''2''  | Заголовок 2-го уровня |
|  ''3''  | Заголовок 3-го уровня |
|  ''4''  | Заголовок 4-го уровня |
|  ''5''  | Заголовок 5-го уровня |
|  ''L''  | Вставить внешнюю ссылку  |
|  ''S''  | Сохранить изменения  |
|  ''P''  | предпросмотр изменений  |
|  ''Y''  | Вставка подписи (только для [[login|авторизовавшихся]] пользователей) |
