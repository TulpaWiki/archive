====== Экспорт статей ======
Отдельная статья вики может быть экспортирована в различные форматы с помощью действия ''export''.

===== Форматы экспорта по умолчанию =====
Следующие опции экспорта доступны по умолчанию:
  - **export_raw** --- «сырой» экспорт
    * возвращает исходник статьи как ''text/plain''.
    * пример: www.tulpawiki.ru/ru:help:export?do=export_raw
    * если вы попытаетесь добавить ''?do=export_raw'' к URL несуществующей (или создаваемой), то будете переправлены на главную страницу.
  - **export_xhtml**
    * возвращает преобразованную статью как простой XHTML без каких-либо элементов навигации
    * пример: www.tulpawiki.ru/ru:help:export?do=export_xhtml
  - **export_xhtmlbody**
    * возвращает только преобразованную к XHTML статью.
    * тэги ''head'' или ''body'' не включаются. Никаких заголовков о типе данных((contenttype headers)) не передаётся.
    * полезен для включения преобразованного содержания в какие-либо другие веб-сайты.
    * пример: www.tulpawiki.ru/ru:help:export?do=export_xhtmlbody

===== Другие форматы экспорта =====
Набор экспортируемых форматов может быть расширен с помощью [[plugins|плагинов]]. Также загляните в [[tips|советы]], чтобы посмотреть на другие форматы экспорта.
